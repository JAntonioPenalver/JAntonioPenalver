## 📁 Proyectos destacados

### 🎮 Slay the Spire Clone - Godot 4.2
Juego de cartas y roguelike desarrollado en equipo durante mi formación educativa. Implementé mecánicas complejas de combate, sistemas de inventario y progresión del jugador.
- **Tecnologías**: Godot 4.2, GDScript
- **Características**: Sistema de batalla por turnos, deck building, progresión de personajes

### 🚀 Asteroids Game - Pygame
Juego clásico de Asteroids desarrollado en Python con Pygame. Incluye lógica de colisiones, movimiento de objetos y sistema de puntuación.
- **Tecnologías**: Python, Pygame, JSONL logging
- **Características**: Física de movimiento, detección de colisiones, scoring system

### 📝 GeneradorDePaginas - Python
Generador de sitios estáticos que convierte markdown a HTML. Implementé un pipeline completo incluyendo parsing, regex, procesamiento de bloques y generación de documentos HTML.
- **Tecnologías**: Python, Markdown, RegEx, HTML Generation
- **Características**: Conversión Markdown → HTML, procesamiento de bloques, pipelineización
